Dieses Beispiel verwendet Media Queries, um die Bilder auf verschiedenen Bildschirmgrößen neu anzuordnen: Für Bildschirme, die breiter als 700px sind, werden vier Bilder nebeneinander angezeigt, für Bildschirme, die kleiner als 700px sind, werden zwei Bilder nebeneinander angezeigt. Für Bildschirme, die kleiner als 500px sind, werden die Bilder vertikal gestapelt (100%).
